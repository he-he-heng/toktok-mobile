# TokTok

모든 세상 사람들이 서로를 마주보면서 하하호호 웃어주는 그날까지, 새로운 사람을 메타버스에서 만나게 해주는 프로젝트 TokTok입니다

## 개요

이 프로젝트는 "모든 세상 사람들이 서로를 마주보면서 하하호호 웃어주는 그날까지"라는 비전을 가지고 제작하게 되었습니다

저희는 메타버스에서 새로운 사람들을 만나게 하기 위한 도전적인 프로젝트를 추진하고 있습니다.
